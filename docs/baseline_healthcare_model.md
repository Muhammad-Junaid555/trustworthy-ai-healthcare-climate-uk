# Baseline Healthcare Diagnostic Model

## Objective

Develop a baseline machine learning model for healthcare diagnostics using publicly available healthcare datasets.

## Dataset

Scikit-Learn Diabetes Dataset

## Model

Random Forest Regressor

## Metrics

- Mean Absolute Error
- R² Score

## Future Improvements

- Explainable AI using SHAP
- Fairness Evaluation
- Bias Detection
- Model Optimization
